═══════════════════════════════════════════════════════════════════

✅ FONDO:

   → Overlay oscuro semi-transparente
   → Efecto de profundidad

✅ CARD:
   → Glassmorphism (vidrio esmerilado)
   → Fondo: rgba(255, 255, 255, 0.08)
   → Blur: 20px
   → Semi-transparente

✅ LOGO:
   → Círculo blanco 100x100px
   → Defensa Civil dentro
   → Sombra pronunciada
   → Animación de entrada

✅ MARCA "GESTISEC":
   → Esquina superior derecha
   → Texto grande y bold
   → "Sistema ITSE" debajo
   → Semi-transparente

✅ INPUTS:
   → Estilo underline (solo línea abajo)
   → Fondo transparente
   → Focus naranja #FF6838
   → Minimalistas

✅ BOTÓN:
   → Gradiente naranja
   → Sombra naranja
   → Hover: sube 2px
   → Efecto de brillo
   → Uppercase

✅ TOGGLE CONTRASEÑA:
   → Ojo en la derecha
   → Click para mostrar/ocultar
   → Color naranja al hover

   http://localhost:4200

4. ¡Verás el nuevo diseño!
